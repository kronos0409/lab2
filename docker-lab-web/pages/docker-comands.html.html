Comandos Especiales Docker

Ver estado de contenedores

Muestra todos los contenedores que están corriendo actualmente.

docker ps

Levantar servicios (Compose)

Crea y levanta los contenedores definidos en el archivo docker-compose.

docker-compose up -d

Limpiar recursos innecesarios

Elimina contenedores, redes e imágenes que no se estén usando para liberar espacio.

docker system prune -a

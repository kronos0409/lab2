Instalación Docker

Configuración paso a paso para desplegar entornos de contenedores en sistemas Linux.

// procedimiento técnico

PASO 01

Actualizar el sistema

Sincroniza los índices de paquetes para asegurar las versiones más recientes.

sudo apt-get update && sudo apt-get upgrade
PASO 02

Instalar dependencias

Configura las herramientas necesarias para descargar paquetes vía HTTPS.

sudo apt-get install apt-transport-https ca-certificates curl gnupg lsb-release
PASO 03

Agregar Llave GPG

Verifica la autenticidad de las descargas con la firma oficial de Docker.

curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg
PASO 04

Instalar Docker Engine

Descarga e instala el motor de Docker y el plugin de Compose.

sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io docker-compose-plugin
